开发背景

Disk Monitor

这是一位摄影师使用 Codex vibe coding 开发的第二款软件，也是第一次把这样一款工具分享给更多用户使用。

在开始这个项目之前，作者从未想过自己有一天真的能够像专业程序员一样做出一款可以运行的 macOS App，并把它发布到网络上，帮助同样需要排查硬盘异常读写问题的人。

这个软件也许仍然存在 Bug，也还有许多值得继续优化的地方。欢迎用户在使用过程中提出建议、反馈问题，帮助它变得更稳定、更实用。
